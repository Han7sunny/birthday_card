## 🎂 Happy Birthday Card 🎂

#### 가족, 친구에게 생일 축하 편지를 보내주세요!

### 기능
  #### 📫 우편함 생성
   + 아이디, 패스워드, 닉네임, 생일 입력을 통해 우편함(계정) 생성

  #### 📩 우편함 편지 작성
   + 편지 작성 시 내용, 작성자, 작성자 인증 비밀번호 입력
   + 작성자 본인 인증을 통해 편지 수정, 삭제 가능
      + 단, 계정주 생일 당일 편지 작성, 수정, 삭제 불가능
   + 우편함에 작성자 이름 노출
  #### 📭 우편함 열람
   + 계정주 생일 당일, 계정주 본인 인증을 통해 우편함 열람 가능
   + 우편함 편지 삭제 가능
  

### 개발 환경
Python, Django Framework, SQLite
